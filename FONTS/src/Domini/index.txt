Potencia.java: Classe que s'ocupa de fer la operacio de potencia per les zones.
Ranking.java: Classe que representa les diferents instancies de ranking per cada tamany disponible de kenken.
Resta.java: Classe que s'ocupa de fer la operacio de resta per les zones.
Suma.java: Classe que s'ocupa de fer la operacio de suma per les zones.
